# Getting started

1. Start DAGonWeb and sign in.
2. Create or upload a DAGonStar workflow JSON file.
3. Add tasks from the palette and give them stable names.
4. Configure Batch commands and use references such as `cat workflow:///prepare/data.txt`.
5. Apply the Inspector configuration; matching links appear automatically.
6. Drag connector handles to add a reference to the target task, or use **Organize canvas**.
7. Changes save automatically.
8. Run the workflow and follow the live graph.
9. Select a completed task on the run graph to inspect its scratch artifacts.

See the [user manual](user_manual.md) for complete guidance.
